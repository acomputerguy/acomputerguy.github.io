The Internet and World Wide Web

Internet (1969)

The Internet is the "worldwide network of interconnected computers and related equipment." In other words, acts as the foundation for a plethora of online applications. Some of these common applications include:

  • Email (1971) [Electronic mail]
    This application is a system of sending electronic mail to accounts on remote machines.
  • Telnet (1972) [Teletype Network]
    The protocol is a tool to let you log in to remote computers.
  • FTP (1973) [File Transfer Protocol]
    The aforementioned protocol can moves files from one computer to another.
  • Gopher (1991) [Protocol for browsing the web]
    This epochal application is the first menu-driven Internet application used to share documents.

World Wide Web (1991)

The World Wide Web is a "system of interlinked hypertext documents connected by the Internet". In short, interconnected web pages with the Internet supporting it. The concept was proposed by Tim Berners-Lee in 1989. Only 2 years later was the Internet available and popularity skyrocket another 2 years after.

The WWW was originally intended to share structured (research) documents between scientists with markups such as paragraphs, headings, lists, and figures with captions. This was done using a web language; HTML (Hypertext Markup Language). Without this, text would appear as long strings on a web page with no easy readability. For example, a 05151990 can be a birthdate but can more understood more closely with markups by 05/15/1990. Or even sections of text being organized such as chapter or topic with headers, paragraph and indentation.

  • Browser
    Popular names of web browsers in this day and age are: Safari, Chrome, Firefox, Internt Explorer, and Opera. In its basic form, it is an application used to procure, present, and transfer information. Some statistics about modern day web is taht 65% of the information on a web page is images and photos (bytes, size wise) and 35 hours of video are uploaded to YouTube daily.[1] Some basic terminology is latency (time it takes to send a packet) and bandwidth (maximum throughput of a logical/physical medium).[2]
  • Lynx
    Lynx is the text web browser. It was designed to show a simple and fast way to get a new product online. As of now, it has an innovative way to talk over the Internet such as turning on an oven from work.[3] With its highly customizable features, it is the oldest web browser in use.[4]
  • Mosaic
    The first GUI browser developed by Mard Andreesen. As of now, it is currently discontinued but its services and mark in Internet history helped shaped the WWW. The Mosiac had revolutionaized the way web browsers acted by "added graphics... ported software from so-called Unix computers."[5] However, by 1998 the web browser was replaced by others.
  • Hypertext
    Hypertext is text that contains links to other text. Another similar term, hypermedia is hypertext containing links to graphics, video and sound. Both terms were coined by Ted Nelson in 1963.

Difference between Internet and WWW

The Internet is the back-end, where servers are running services. The WWW is the front-end, where web sites are often displayed with interactive behavior between the site and the user.
