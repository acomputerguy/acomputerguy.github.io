On the first day of work, I like to configure my web browser and email to get more work done efficiently. Keyword bookmarks, email filtering, and web browser shortcuts will be discussed.

Keyword Bookmarks in Web Browser

I usually prefer Chrome, but the guide will be on Firefox. For the company intranet, I go through this process -

Example: wikipedia

>Right-click on the search bar in Wikipedia

>Select Add a keyword for this search...

>In the Keyword bar, I typed wiki

>In address bar, type in wiki <topic> and hit enter!

Sometimes, this doesn't work for every website. The web browser doesn't properly format the query to their database, so there is a more manual set of steps to follow:

>Pull up the search engine results with a random string of characters (not a page)

>Copy the URL

>Click on Show your bookmarks icon in the top right

>Click on Show all bookmarks

>Right-click on Bookmarks Toolbar

>From the drop-down, click on New Bookmark...

>Fill out the information as you see fit and paste the URL in the Location section

>Replace the random string of characters from before with %s and hit Add

The %s means the argument is taken in as a string after the keyword in the URL, then submitted as part of the query to pull up results from their database.

Why use bookmark shortcuts, wait for the page to load, then type in the search engine when you can do all 3 at once from the address bar? Doing so saves me about 10 seconds. While in the short-term, seconds may seem like nothing but in the long run, this technique helps me save minutes, even hours. As a frequent computer user, I find this to be highly useful. This is another step closer to being the computer guy who only uses the keyboard to navigate through a computer!

Email filtering

I've noticed either Microsoft Outlook or Gmail is preferred, but I'll be talking about gmail here because I wrote a script to assist with filters. Usually once you start the job, you're bombarded with emails so during first week's downtime, I like to filter certain categories into certain folders. For filtering emails such as various project topics or miscellaneous (news, events), I go through this process.

>Click on the arrow on the right side of the search bar

>Filter with given properties; From, To, Subject, Has the words, Doesn't have

>For my example, I chose From my email address and To my email address, subject test subject, and has the words this is a test. This is for test cases when I send myself email. When finished, click on Create filter with this search >>

>Click on Apply the label check box and from the drop-down options and click on New Label

>For my example, I typed test label. Click on Create

>Go back to the Inbox

>Type in the search bar, label:<name of label>, where the name of my label is label test

>In the toolbar at the top, click on the checkbox

>Click on the archive button

I don't want to show only archived mail then click the archive button every single time when this can be automated with a script.

Instead of having to continuously archive in the last 4 steps, I created a script using Google Apps Script, which is in JavaScript with Google's own suite of libraries.

My approach to Google Apps Script:

You can use a built-in feature to create multiple inboxes, but they exist with extra panes taking up part of the screen.

Shout out to HubSpot for making a simple guide!

I don't want any extra space taken up on the screen, I would rather be able to click on the inbox and show the messages specific to a given category.

Gmail doesn't have a way to create multiple inboxes that do not show up on a pane. I want to avoid taking up space on the screen so my script will have similar properties as gmail "inbox", which is done through labeling and archiving in the background instead of foreground. I could add some code for automatic labeling, but why reinvent the wheel?

Having no background in JavaScript, programming a script to accomplish this task was an enjoyable experience.

Thanks to Google Apps Script for their beautiful documentation and query examples!
